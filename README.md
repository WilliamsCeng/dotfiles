### Description
dotfiles 是指 Linux 工具常用 .(dot)开头的文件来进行 个性化的配置

为了在不同的机器上同步这些配置，可以采取dotfiles的策略

如下：

dotfiles文件夹+ symlinks + git 版本控制 + bash 脚本自动 (在特定的目录下 生成 symlinks 完成工具的自动配置)
